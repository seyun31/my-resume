## **🙋🏻‍♀️ It's Me**
**안녕하세요, 불확실함을 성장의 발판으로 삼는 프론트엔드 개발자 임세윤입니다.**

새로운 문제를 배움의 기회로 삼아 끝까지 해결하며 성장해왔습니다.


최근 **Next.js 기반 CSR/SSR 혼합 아키텍처**를 적용해 **FCP를 50%, 서버 응답 지연을 56% 단축**했습니다.

또한 **Sentry와 Google Analytics**를 활용해 Web Vitals(LCP, CLS, FCP, INP, TTFB)과 사용자 데이터를 분석하고, 이를 바탕으로 **UI/UX를 개선**했습니다.

TypeScript에서는 `any` 타입을 배제하기 위해 **strict 타입 검사와 ESLint 규칙을 강화**하여 빌드 단계에서 오류를 조기 탐지하도록 환경을 구성했습니다.

**Tailwind CSS v4**와 **Three.js**를 활용해 최신 웹 기술을 적극 반영하고, 인터랙티브하고 접근성 높은 UI를 구현했습니다.


사용자에게 **편리하고 즐거운 경험**을 제공하기 위해 끊임없이 실험하고 개선하는 개발자입니다.
