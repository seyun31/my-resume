### ⚙️ [Commit Helper]
[👾GitHub](https://github.com/seyun31/commitHelper)

**📘 서비스 개요**  
- Git의 staged 변경 사항을 자동 분석하여 **Conventional Commits 규칙에 맞는 커밋 메시지를 추천**하는 대화형 CLI 도구입니다.  
- 개발자가 일관된 커밋 컨벤션을 유지하고, Git에 익숙하지 않은 팀원도 손쉽게 협업할 수 있도록 지원합니다.

**💡 담당 파트**
- `simple-git` 기반 **Git diff 분석 엔진 개발:** staged 파일의 변경 사항을 라인 단위로 분석  
- **커밋 타입 자동 분류:** 추가/수정/삭제 비율 분석으로 7가지 커밋 타입 예측  
- **대화형 CLI 인터페이스:** `inquirer` list prompt 활용, AI 추천 메시지 → 직접 수정·확인 플로우 구현  
- **Git 연동 모듈:** `--cached`, `--name-only` 옵션 기반 비동기 파일 처리  
- **Jest 유닛 테스트 작성** 및 **GitHub Actions CI/CD 파이프라인 구축**  
- **NPM 패키지 배포 및 오픈소스 공개**

**🛠️ Stacks & Tools**  
- `TypeScript 5.8.3` · `Node.js 18+` · `simple-git 3.27.0` · `inquirer 12.6.1`
